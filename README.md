# 米什金互动学习 · Luffa SuperBox 小程序

把米什金《货币金融学》第 4 篇(中央银行与货币政策)+ 第 6 篇(货币理论)中的核心机制(货币乘数 / 利率平价 / IS-LM)做成可在 Luffa SuperBox 上跑的**互动小程序**——基于学习科学六原理设计,把"教材的冷理论"变成"今天的市场上立刻验证的活教材"。

> **当前状态**:规划阶段已完成(12 份文档),Taro PoC v0 已编译通过(448K,可直接导入 SuperBox IDE)。下一步是真机测试 + 内容流水线启动。

## 三章 MVP 的反预期震撼

| 章 | 模拟器 | 反预期触发点 |
|---|---|---|
| **第 14 章 货币乘数** | 4 滑块(MB/r/e/c)+ 4 历史快照 | **2010.1 QE 后:乘数从 8.7 砸到 1.2**——Fed 印万亿 M2 却几乎没动 |
| **第 17 章 套息交易** | 4 滑块 + 5 历史快照 + 风险标签心跳 | **2024.8 USD/JPY 三周从 162→142**——10x 杠杆 -100%,10 年 carry 收益 3 周亏完 |
| **第 20 章 IS-LM 沙盘** | 4 滑块 + Canvas 双线 + 5 历史快照 | **2010 流动性陷阱:LM 推右多少,Y* 都不动**——Fed "推不动弦" |

每个反预期都有**预测先行 → 用户被打脸 → 揭示**三段闭环,这是学习科学"反预期 + 测试效应"的核心。

## 关键技术决策

- **平台**:Luffa SuperBox(微信小程序克隆架构,API = `wx.*`)
- **前端**:Taro 4 + React 18 + TypeScript 5
- **图表**:Canvas 2D(IS-LM 沙盘);SVG 不被 SuperBox 支持
- **后端**(规划):FastAPI + Postgres + pgvector + Redis(详 [docs/05](docs/05_技术架构选型_v0.md))
- **数据**:FRED 18 因子 + Postgres schema(详 [docs/11](docs/11_数据Pipeline规格_v0.md))
- **AI**:Claude Sonnet/Opus 6 个 prompt(详 [docs/09](docs/09_AI_Prompt模板集_v0.md))
- **不做**:代币 / Web3 激励 / 合规层(产品决策)
