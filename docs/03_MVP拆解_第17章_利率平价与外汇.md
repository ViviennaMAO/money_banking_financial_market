## 2. 套息交易实验室(P0 模拟器) 🔧(框架 95% 复用,公式 + 字段全换)

### 2.1 数学模型 🆕

**抛补利率平价(CIP)**:

```
F = S × (1 + i_d × τ) / (1 + i_f × τ)
```

**套息交易回报近似公式**(站在 USD/JPY 视角,借 JPY 投 USD):

```
总回报 ≈ 利差 carry + 汇率变动 ΔS/S
       = τ × (i_USD - i_JPY) + (S_τ - S_0) / S_0
```

**Breakeven 条件**(汇率允许的最大不利变动):

```
ΔS/S 最差 = -τ × (i_USD - i_JPY)
```

意思是:carry 收益吃掉多少,汇率才能跌多少而不亏。这是套息交易的"安全边际"。

## 12. 风险与开放问题 🆕

1. **数据成本**:汇率分钟级数据 + 远期点的免费源覆盖度差,可能要付费(Bloomberg ≈ 2k USD/月)。MVP 建议:即期免费源 + 远期采集每日 1 次即可
2. **杠杆教育的责任边界**:模拟器演示"10x 爆仓"具有明确教育价值,但务必显眼标注"教学用途,非投资建议"。考虑加一个"风险测试":连续触发 3 次 margin call 才允许继续玩高杠杆
3. **三元悖论的政治敏感**:中国央行干预的内容尺度(参考第 14 章已有的中美对比尺度)
4. **2024.8 案例的时效性**:这个案例 1-2 年内还很鲜活,3-5 年后需要换新案例。设计内容时要考虑"案例过期"机制
5. **CIP 失效的复杂性**:学术上 CIP 在 2008 后系统性失效(post-GFC anomaly),要不要在 MVP 里讲?(我倾向作为 §3.4 误区诊断的进阶素材,不放主线)

## 15. 模板验证小结 🆕(本章作为第二案例的特殊节)

跑完两章可以下三个判断:

1. **学习时刻链结构稳健**:§0-§14 的 14 节框架同时覆盖了"宏观存款机制"和"国际金融"两个差异极大的主题,说明模板不是为第 14 章定制的
2. **复用率比 v0 估的低,但仍可用**:实际节省 33%(7/21 人日),不是 60%。第三章(IS-LM)预计节省 40-45%,因为框架更熟、误区库可以参考前两章
3. **真正的瓶颈是"内容研究 + 题库设计"**,而不是"重写交互"。下一步如果要再加速,要做的是建一个"内容生产流水线"(教研 + 题库 + 误区共建),而不是再加交互模板

**建议**:第三章(IS-LM)做完后,把三章的误区库(8+8+8 = 24 条)合并成"全 MVP 误区底库",做一次跨章一致性 review。这会暴露教学盲点(比如 IRP-01 和 MM-03 都涉及"常数误区",可以合并成一个跨章微课)。
