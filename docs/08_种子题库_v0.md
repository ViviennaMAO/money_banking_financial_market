### 2.1 货币供给三位参与者(2 道)

```
[ch14-q012] ⭐ 概念辨析
教材定义货币供给过程的"三位参与者"中,下面哪一个不属于?

A. 中央银行    B. 商业银行    C. 储户    D. 财政部 ✓

解析:三位参与者 = 央行(决定 MB 和 r)、银行(决定 e)、储户(决定 c)。
财政部不在这个框架内——其支出影响 IS 但不直接进货币创造方程。
误区:MM-04、MM-08

```

```
[ch14-q013] ⭐⭐ 政策推理
"三位参与者各管一个变量"框架下,下面哪种情境**最难**通过央行单方面操作改变 M2?

A. 商业银行集体不愿放贷,e 上升   ✓
B. 储户提现减少,c 下降
C. Fed 启动 OMO 买债
D. 法准率 r 调整

解析:e(超额准备金率)由银行自主决定,央行只能通过 IORB 间接影响。
2008-2014 期间 e 从 0.04% 飙到 80%,Fed 怎么 QE,m 都被压死。
误区:MM-03、MM-08
关联:§2.4 「2010.1 QE 后」历史快照
```

### 2.2 美联储资产负债表 +「美元流动性公式」(3 道)

```
[ch14-q015] ⭐⭐ 数据解读
2024.4 关键数据:Fed 总资产 $7.4T,TGA(财政部一般账户)$0.95T,
ON RRP(隔夜逆回购)$0.45T。按教材"美元流动性 = Fed 资产 - TGA - ON RRP":

A. $5.5T    B. $6.0T ✓    C. $7.4T    D. $8.8T

解析:7.4 - 0.95 - 0.45 = $6.0T。这个公式反映 Fed 资产负债表中"真正在金融市场流通"的流动性。
TGA 是财政部存在 Fed 的钱(暂时锁住),ON RRP 是货币基金等把现金放回 Fed(也锁住)。
误区:MM-02
关联:第 4 篇 §14 讨论课题
```

```
[ch14-q016] ⭐⭐⭐ 政策推理
2023.6 美国债务上限解决后,财政部启动"补 TGA"行动,3 个月内 TGA 从 $0.04T 升到 $0.6T。
按 §2.2 公式,这对美元流动性的影响:

A. 增加 $0.5T    B. 减少 $0.5T+ ✓    C. 不影响    D. 取决于 ON RRP

解析:TGA↑ 直接吸走流动性。这就是为什么 2023.6-9 美国股债承压、信用利差走阔——
"隐形紧缩"约 $0.5T,等同一次 mini-QT。
误区:MM-08、MM-01
关联:第 15 章 货币政策工具
```

```
[ch14-q017] ⭐⭐⭐ 概念辨析
2023.3 SVB 倒闭后 Fed 创设 BTFP(Bank Term Funding Program),银行用国债**面值**(而非市价)做抵押借现金。这个操作影响 Fed 资产负债表哪一端?

A. 资产端 ↑(贷款增加),负债端 ↑(准备金增加)✓
B. 只资产端变化
C. 只负债端变化
D. 资产负债表大小不变,内部重组

解析:Fed 借出 → 资产端"贷款"项 ↑;银行拿到现金存回 Fed → 负债端"准备金"↑。
教学点:Fed 任何流动性投放都同时扩资产和负债,资产负债表是"同步"扩张的。
误区:MM-08
关联:§2.4 模拟器:增加 MB 后看 M2 反应
```

### 2.4 货币乘数(3 道)

```
[ch14-q021] ⭐⭐⭐⭐ 政策推理
Fed 把 IORB(超额准备金利率)从 4.40% 上调到 5.40%,在其他不变情况下,主要影响是:

A. m 上升,M2 增加
B. m 下降,M2 减少 ✓
C. MB 上升,M2 增加
D. r 上升,M2 减少

解析:IORB↑ → 银行更愿意把准备金留 Fed 吃利息 → e ↑ → 分母变大 → m ↓ → 给定 MB,M2 ↓。
这是 Fed 紧缩货币政策的"利率走廊地板"工具。
误区:MM-07
关联:§2.4 模拟器调高 e 看 m 变化
```

```
[ch14-q023] ⭐⭐⭐ 数据解读
2008-2014 Fed 资产负债表从 $0.85T 扩到 $4.5T(增 5 倍多),但 M2 只从 $7.7T 增到 $11.7T(+50%)。教材给出的最关键解释是:

A. 财政赤字限制了 M2 扩张
B. 货币乘数 m 大幅下降(8.7 → 3.0 左右)✓
C. 通胀预期锚定让人不愿持币
D. 美元国际化稀释了 M2

解析:e 飙升 → m 暴跌,主导了 MB 与 M2 之间的 disconnect。教学要点:**MB 与 M2 通过 m 这个变量耦合,而 m 不是常数**。

```

```
[ch14-q024] ⭐⭐⭐ 概念辨析

C. 货币乘数公式不同
```

### 2.4.5 制度演进:IORB 合并(1 道,填 q025 reserved gap)🆕 v0.1 补丁

> **关键事实**:Fed 在 **2021.7.29** 把 IOER(超额准备金利率)+ IORR(法定准备金利率)合并为单一的 **IORB**(Interest on Reserve Balances)。米什金教材撰写时未做此修改,讲义和模拟器以**现行 Fed 工具**(IORB)为准,但题库专门设题让学生看到这一制度演进。

```
[ch14-q025] ⭐⭐⭐ 概念辨析(教材 vs 现实)

米什金教材中讨论"超额准备金利率(IOER)"和"法定准备金利率(IORR)"两个工具。
但 2021.7.29 起 Fed 实际把它们合并为单一"准备金余额利率(IORB)"。原因是?

A. Fed 想简化政策工具(简化论)
B. 美国 2020.3 起法准率 r = 0,IORR 失去意义,合并为 IORB 是制度的自然演进 ✓
C. 适应数字货币(CBDC)
D. 监管要求

解析:r = 0 意味着所有银行准备金都是"超额"(没有"法定"基准),IORR 和 IOER 数值上无差别。
2021.7 合并为 IORB 是 Fed "准备金充足制度"(ample reserves regime)的配套调整。
**教学点**:教材的工具集会过时,现实工具集在演进——
这是 §13 误区库「教材 vs 现实」差距的最直接证据,也是跨章微课《模型的边界》的现实素材。
误区:MM-07 + 教材-现实差距
关联:跨章微课《模型的边界》、[02 §13] MM-07、第 15 章 货币政策工具
🔓 解锁徽章「教材现实差距读者」
```

### 3.2 IS / MP / LM 曲线(3 道,关键!)

```
[ch20-q013] ⭐⭐ 概念辨析
教材明确指出"IS 曲线下斜"的根本原因是:

A. 利率上升 → 投资 I 下降 → 总产出下降 ✓
B. 利率上升 → 消费下降
C. 利率上升 → 政府支出下降
```
