# 内容流水线 v0

> **目的**:把"三章 MVP 需要的 90 道题 + 50 条新闻 + 24 条误区 + 3 个跨章微课"从空气变成可执行的内容生产计划。

## 1. 24 条误区合并 → 4 组跨章模式

把第 14/17/20 章的 24 条误区放到一起看,会浮现 4 组共享底层错误模式:

### 模式 A · 「参数当常数」(最高频,3 章都中)

| 误区代码 | 章 | 表面错误 | 共同底层 |
|---|---|---|---|
| MM-03 | 14 | "m 是常数" | 把动态参数看成死数 |
| ISLM-02 | 20 | "LM 总上斜" | 同上 |
| IRP-01 | 17 | "高息货币必升值" | 同上 |

**底层**:用户看到金融模型时,直觉上会把模型参数当作"自然常数"。但**金融的核心特征恰恰是参数本身在变化**,且参数变化往往是机制崩塌的信号。

→ **跨章微课 1 · 《参数即机制》**(详见 §2.1)

### 模式 B · 「政策万能」

| 误区代码 | 章 | 表面错误 | 共同底层 |
|---|---|---|---|
| MM-08 | 14 | "QT 销毁钱 / QE 印钱" | 不知政策有传导边界 |
| ISLM-04 | 20 | "货币政策永远有效" | 同上 |
| IRP-04 | 17 | "央行干预万能" | 同上 |

**底层**:把政策工具当成线性 input → output。

→ **跨章微课 2 · 《政策的无效区》**(§2.2)

### 模式 C · 「模型成立条件」

| 误区代码 | 章 | 表面错误 | 共同底层 |
|---|---|---|---|
| MM-01 | 14 | "印钱必通胀" | 没问"什么条件下成立" |
| ISLM-03 | 20 | "财政必挤出" | 同上 |
| IRP-05 | 17 | 混淆 covered/uncovered IRP | 同上 |

→ **跨章微课 3 · 《模型的边界》**(§2.3)

### 模式 D · 「量价混淆」

| 误区代码 | 章 | 表面错误 | 共同底层 |

| MM-02 | 14 | "M2 = MB" | 混淆"量"(规模)和"价"(乘数/利率) |

## 2. 三个跨章微课(完整内容)

每个微课结构:`Hook(60 秒动画) → 三章共点 → 总结 → 验证测验(2-3 道)`。微课只对**完成 MVP 三章的用户**开放,作为高级彩蛋。

### 2.1 微课 1 · 《参数即机制》

**Hook 动画(60 秒)**:
屏幕上出现一条平滑曲线 → 用户拖参数 → 曲线突然在某个临界点**断裂、重新组装成完全不同的形态**。配文:"你刚刚看到的不是数字变了,是机制本身换了。"

**三章共点拼图**:

```
┌─────────────────────────────────────────┐
│  第 14 章:m = 8.7 → 1.2                 │
│  不是"乘数变小了",而是"派生失灵了"      │
├─────────────────────────────────────────┤
│  第 20 章:LM 上斜 → LM 水平             │
│  不是"斜率变了",而是"利率工具失效了"    │
├─────────────────────────────────────────┤
│  第 17 章:高息升值 → 高息贬值           │
│  不是"方向反了",而是"主导力量换了"      │
│  (carry → 风险溢价)                    │
└─────────────────────────────────────────┘
```

**总结**:**当一个金融参数显著偏离历史常态,你看到的不是"异常数字",而是底层机制的相变(phase transition)。**

**验证测验**(3 道,完整答对解锁徽章「机制读者」):

```
1. 下面哪些情境是"参数即机制"的体现?(多选)
   ☐ A. 2008 后 m 从 8 砸到 1
   ☐ B. 通胀从 2% 升到 9%
   ☐ C. LM 从上斜变水平
   ☐ D. 高息货币从升值变贬值
   ☐ E. M2 增速从 5% 升到 8%
   答案:A、C、D(B 和 E 是参数变化但同机制内)

2. 你看到 USD/JPY 利差扩大但日元在升值,最可能的解释是?
   ☐ 数据错了
   ☐ IRP 失效
   ☐ 风险溢价主导取代了 carry 主导(机制相变)✓
   ☐ 央行干预

3. 简答(自由文本):用一句话告诉朋友
   "为什么金融模型的参数比模型本身重要"
   AI 评分:看是否提到"相变"、"主导机制切换"或等价表述
```

### 2.2 微课 2 · 《政策的无效区》

**Hook 动画**:
屏幕上 Fed 主席不停按"加息""降息""QE""QT"按钮 → 经济曲线却不动 → 配文:"为什么有时按了没反应?"

**三章共点拼图**:

```
┌─────────────────────────────────────────┐
│  第 14 章:超储锁住 → QE 不传导到 M2     │
│  机制 1:m 趋近 1                       │
├─────────────────────────────────────────┤
│  第 20 章:ZLB → 货币政策水平段失效       │
│  机制 2:利率没有下行空间                │
├─────────────────────────────────────────┤
│  第 17 章:三元悖论 → 央行不能同时       │
│  控汇率 + 控利率 + 开放资本账户          │
│  机制 3:三选二必有一弃                  │
└─────────────────────────────────────────┘
```

**总结**:**每种政策都有"无效区",而无效区的存在恰恰反映了政策传导是有边界的物理过程,不是按钮魔法。**

**验证测验**(2 道):

```
1. 情境题:经济衰退、利率已 0.05%、银行超储 80%。
   下面哪个政策最有效?
   A. 央行再降息 25bp     B. QE 再扩 1 万亿
   C. 财政直接转移支付     D. 准备金率降到 0
   答案:C(只有财政能绕过 m 失灵和 ZLB)

2. 多选:中国央行想"既稳汇率又用独立货币政策又允许资本自由流动",
   按三元悖论必须放弃哪一个?选最现实的两条出路。
   ☐ A. 资本管制(放弃自由流动)✓
   ☐ B. 浮动汇率(放弃稳汇率)
   ☐ C. 完全放弃货币政策独立性
   ☐ D. 三个都放一点(部分约束)✓
```

### 2.3 微课 3 · 《模型的边界》

**Hook 动画**:
"印钱必通胀" "财政必挤出" "套息稳赚" 三句标语先依次亮起,然后依次被划掉,最后亮起一句:"每个模型都有它的成立条件。"

**三章共点拼图**:

```
┌──────────────────────────────────────────┐
│  "印钱必通胀" — 货币数量论                │
│  成立条件:V 稳定 + m 稳定                │
│  失效场景:2010-2014 美国(V↓ + m↓)     │
├──────────────────────────────────────────┤
│  "财政必挤出" — 标准 IS-LM                │
│  成立条件:LM 上斜段                      │
│  失效场景:流动性陷阱(LM 水平)          │
├──────────────────────────────────────────┤
│  "CIP 永远成立" — 抛补利率平价            │
│  成立条件:无套利 + 美元充足              │
│  失效场景:2008、2020 流动性危机          │
└──────────────────────────────────────────┘
```

**总结**:**金融教科书里的每个"必然"句式,后面都隐含一组成立条件。学会问"什么条件下成立"比学会公式更重要。**

**验证测验**(2 道):

```
1. 让你判断下面命题"在什么条件下成立":
   "央行降息会刺激经济增长"
   要求:写出至少 3 个隐含条件(自由文本,AI 评分)
   参考答案:
   - 银行愿意把超储释放出来(否则被 IORB 吸收)
   - 实体部门有信贷需求(私人去杠杆期不成立)
   - 当前不在流动性陷阱(否则货币政策失效)
   - 通胀预期不会反向急升(否则实际利率反升)

2. 第 10 题(三章合一)的升级版:为什么"米尔顿 · 弗里德曼:
   '通胀永远是货币现象'"既对又不对?
   选最准确的解释:
   A. 弗里德曼错了
   B. 长期对、短期错(因为短期 V 和 m 不稳)✓
   C. 只对美国成立
   D. 在 IRP 框架下不成立
```

```yaml

type: data_interp  # concept / formula / data_interp / policy / counter_intuitive
objective: analyze # 布鲁姆层级

  B: "..."
  C: "..."
  D: "..."

distractor_logic:  # 关键!每个错项绑定误区

  C: MM-03
  D: noise

**关键设计**:**错项不是随机噪声,每个都对应一个具体误区**。这是 §3.4 误区诊断能精准定位的前提。

```
[ch14-q011]  ⭐⭐  概念辨析
"准备金"和"存款准备金率"是同一个东西吗?
A. 是  B. 准备金是金额,准备金率是比例 ✓
[误区映射: MM-04(降准必增 M2)]

[ch14-q014]  ⭐⭐⭐  数据解读

  - 2010: 1.0(谷底)
  - 2024: 5.4
2010 → 2024 期间 m 上升,主要是什么变化?
A. r 下降 (r 在 2020 已经为 0,变化不大)
B. e 下降(QT 减少超储,银行愿意贷款)✓

D. MB 下降(其实 MB 没大幅下降)

[ch14-q018]  ⭐⭐⭐⭐  政策推理
2023.3 SVB 倒闭后 Fed 创设 BTFP(Bank Term Funding Program),
允许银行用国债面值(而非市价)做抵押借现金。
这个工具对货币乘数的影响?
A. 直接增加 MB,所以 M2 上升
B. 不改变 r、e、c,所以 m 不变,但 MB 上升 → M2 上升 ✓
C. 让 e 上升,m 下降
D. 不影响 m 也不影响 M2(只是流动性互换)
[误区映射: MM-01、MM-08]

[ch14-q022]  ⭐⭐⭐  概念辨析
中国"降准"(下调 RRR)和美国"QE"在货币创造机制上,主要区别是:
A. 降准动 r,QE 动 MB(c、e 不动)✓
B. 降准动 e,QE 动 c
C. 降准动 c,QE 动 r
D. 两者机制完全相同
[误区映射: MM-05]

[ch14-q030]  ⭐⭐⭐⭐⭐  反常识
日本央行从 1990s 起 m 长期低位徘徊在 3-5,
但 1980s 巅峰时 m 曾达 12。
日本 m 的长期下台阶反映了什么?(多选)
☐ A. 日本人持现金比例 c 上升(老龄化)✓
☐ B. 银行长期超储(零利率持续)✓
☐ C. 影子银行替代了传统银行(部分)
☐ D. 准备金率 r 上调(其实没有)

### 3.4 出题工作流

```
教研主笔列出 30 个目标(章节×误区×难度) → 每个目标出 1 题
  → AI(Claude Opus)根据元数据生成草稿
  → 教研编辑改写,设计错项
  → 题库编辑映射误区代码 + 写解析
  → 用户内测(5-10 个种子用户答)
  → 答错率 > 70% 或 < 10% 的题重做(太难或太简单)
  → 上线
```

**预算**:每道题 30-45 分钟(包括 AI 草稿 + 编辑 + 校对)= 30 题 × 0.5 小时 = 15 小时/章 = 2 人日/章。

## 4. 新闻三联单内容池

### 4.1 触发场景分类

不是所有新闻都值得做三联单。只在以下 4 类触发:

| 触发类 | 频率 | 例子 |
|---|---|---|
| **决议日** | 月度 | FOMC、ECB、BOJ、PBOC 利率决议 |
| **数据日** | 月度 | CPI、NFP、PCE、PMI 发布 |
| **危机/事件日** | 不定期(高优先) | SVB 倒闭、英国迷你预算危机、瑞郎脱钩 |
| **政治/制度日** | 季度 | 债务上限、财政预算、央行人事 |

### 4.2 10 条真实新闻种子(覆盖三章)

```
[N01] 2024.5  Fed 6 月起放缓 QT (已在 02 §4.2)         → 第 14 章
[N02] 2024.7  BOJ 加息 + Powell 转向 (已在 03 §4.1)    → 第 17 章
[N03] 2024.10 美财政赤字 vs Fed 紧缩 (已在 04 §4.1)    → 第 20 章
[N04] 2023.3  SVB 倒闭 → BTFP 创设                     → 第 14 章
[N05] 2024.X  中国央行降准 → M2 vs 信贷需求            → 第 14 章
[N06] 2022.9  英国 Truss 迷你预算危机(GBP/Gilts 风暴)  → 第 17 + 20 章
[N07] 2015.1  瑞士央行脱钩 EUR/CHF 上限                → 第 17 章(经典案例)
[N08] 2008.10 雷曼倒闭 → 信用利差崩 + LM 进陷阱        → 第 14 + 20 章
[N09] 2020.3  疫情双扩张(财政 + 货币)                 → 第 20 章
[N10] 2024.X  中美利差倒挂 + 套息反向                  → 第 17 章
```

每条按 [02 §4.1](02_MVP拆解_第14章_货币乘数.md) 模板展开:**新闻 50 字 → 看板信号 3 个 → 教材原理 + 章节定位 → 渐进追问 3 层**。

```

    ↓

```

### 4.4 AI 草稿 Prompt 模板

## 8. 风险与开放问题

1. **教研主笔难找**:既懂金融又会教的人稀缺。备选:招金融背景应届生 + 资深金融人士做顾问审稿
2. **AI 草稿质量参差**:第一个月新闻三联单可能 50% 需要重写。建议第一个月主笔全程参与,3 个月后再让 AI 主导
3. **答错率数据需要 1000+ 题次才稳定**:第一版题库一定有"太难"和"太简单"的偏差,需要内测 + 上线后快速迭代
4. **跨章微课依赖三章都完成**:用户必须先学完 14/17/20 才解锁。冷启动期(MVP 第 1-2 个月)可能只有少数用户能解锁,要观察是否值得做
5. **新闻时效性**:50 条种子里有 10 条是 2008/2015/2022 历史案例(可永久用),其余 40 条是当年新闻,1-2 年后需要换。设计内容时**优先选历史经典案例**(永久价值高),时事新闻只放 30-40%
