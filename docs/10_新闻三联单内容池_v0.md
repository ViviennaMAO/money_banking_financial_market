## 5. 内容质量自检清单(每条上线前过)

```
[ ] 标题 ≤ 25 字
[ ] 新闻原文浓缩 ≤ 50 字
[ ] 看板信号必给 3 个,**有具体数值**(不准编造)
[ ] 章节定位精确到 §,有跳转链接
[ ] 渐进追问 3 层:L1 直觉(单选)/ L2 机制(自由文本)/ L3 反事实
[ ] 如涉及具体投资品 → 加"教学用途,不构成投资建议"
[ ] 涉及中国相关时,机制对比为主,不做政策评论
[ ] 教研注解给出"何时推 / 推给谁"
[ ] 有 expires 字段(时事案例)或永久(历史案例)
```
